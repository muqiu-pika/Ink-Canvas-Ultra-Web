# 更新日志

## 最新版本：7.0.2

### V7.0.2
- 更新软件版本号为 V7.0.2
- 微调部分窗口 UI，删除部分功能的重复逻辑
- 优化目录结构，便于项目管理（MainWindow / Windows / Helpers / Models 等）
- 重新设计部分窗口的 UI 界面，并修复一些小 Bug
- 视频展台侧栏添加一键清除所有白板内容的按钮
- 修复软件编译缺少包 ID 的问题
- 尝试修复蓝牙断连后继续触发的潜在阻塞问题
- 初步拆分 MainWindow 文件，删除多余的设置页面
- 为 GitHub Actions 编译出的 zip 程序文件添加版本号
- 优化设置界面设计，添加侧边导航栏

### V7.0.1
- 更新软件版本号为 V7.0.1

### V6.2.1
- 更新软件版本号

- 删除墨迹识别功能中"非等粗线条"的提示输出

### V6.2.0
- **修复安全问题**：
  - Undo/Redo 越界：添加边界检查，无效时返回 null
  - Null 引用防护：所有 foreach 前增加 null 检查，字典使用 TryGetValue
  - 路径穿越攻击：解压文件时验证路径，防止 ../ 跳出目录
  - 非法文件名：添加 SanitizePathSegment 方法清理特殊字符
  - Bitmap 内存泄漏：使用 using 语句确保资源释放
  - 重复 Shutdown：移除冗余调用，只保留一处
  - 热键 keymap 泄漏：注销时从字典中移除对应项
  - 重启路径获取：添加多种备用方案（进程模块、命令行参数等）
- 修复浮动栏位置 bug
- 添加摄像头画面插入错误时的检测
- 修复部分编译警告
- 优化 COM 接口检查逻辑
- 软件全局粘贴快捷键改为 Ctrl+Shift+V
- 修复 PPT 播放模式进入白板模式时双指手势遵循常规手势设置的问题
- 不再受 PPT 演示模式下的手势设置限制
- PPT 翻页手势在白板模式下禁用，避免冲突

### V6.0.4
- 修复 PPT 放映退出时，画板批注笔迹被替换的 bug
- 修改截屏逻辑
  - 启动截屏：按 Alt+C 或在浮动栏点击截图按钮后，屏幕变暗显示十字光标
  - 选择区域：按住鼠标左键/触摸屏幕拖动选择要截图的区域
  - 确认截屏：松开鼠标左键/抬手完成区域选择
  - 操作选项：弹出截图操作窗口，提供：插入到当前画板、保存到文件夹、加入照片列表、取消
- 调整视频展台侧边栏中缩略图清晰度
- 添加触摸窗口注册（RegisterTouchWindow API），支持电容屏、电阻屏、数位屏、手写屏、红外触摸屏等设备
- 修复初始设置页面中按钮名称错误
- 修复 UI 线程阻塞问题
- 修复退出 PPT 模式后会替换掉画板批注笔迹的问题
- 调整默认设置参数
  - 直线延长阈值默认和推荐均为 120，灵敏度默认和推荐为 0.8
  - 浮动工具栏 UI 缩放默认和推荐为 80
  - 矩形、三角形"等粗线"（无压力）默认和推荐均为开启
  - PPT 侧边翻页箭头按钮、侧边页码按钮默认和推荐均为关闭
  - 自动黑屏截图默认关闭，推荐也为关闭
  - 记住并提示上次播放位置默认和推荐均为开启
  - 崩溃时静默重启默认和推荐均为开启
  - 黑板缩放比例默认和推荐均为 80
- 优化双指移动时的卡顿，减少画面抖动
- 修复 PPT 放映模式无法自动进入的 bug
